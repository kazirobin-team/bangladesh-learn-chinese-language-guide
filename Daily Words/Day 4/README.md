
**৩৭. 喝 (hē)**  
**Pinyin:** Hē  
**অর্থ:** পান করা (drink)  
**বাক্য:** 我想喝水。  
**Pinyin:** Wǒ xiǎng hē shuǐ.  
**অর্থ:** আমি পানি খেতে চাই।

**৩৮. 和 (hé)**  
**Pinyin:** Hé  
**অর্থ:** এবং / সাথে (and; with)  
**বাক্য:** 爸爸和妈妈都在家。  
**Pinyin:** Bàba hé māma dōu zài jiā.  
**অর্থ:** বাবা এবং মা দুজনেই বাসায় আছেন।  
**বাক্য:** 他和王先生说话呢。  
**Pinyin:** Tā hé Wáng xiānsheng shuōhuà ne.  
**অর্থ:** তিনি ওয়াং স্যারের সাথে কথা বলছেন।

**Pinyin:** Lǐ xiǎojiě hěn piàoliang.

**৪০. 后面 (hòumiàn)**

**অর্থ:** পেছনে / পরে (behind; later)

**৪১. 回 (huí)**  
**Pinyin:** Huí  
**অর্থ:** ফিরে যাওয়া / বার (go back; number of times)  
**বাক্য:** 我八点回家。  
**Pinyin:** Wǒ bā diǎn huí jiā.  
**অর্থ:** আমি রাত আটটায় বাসায় ফিরব।  
**বাক্য:** 我去过一回北京。  
**Pinyin:** Wǒ qù guo yī huí Běijīng.  
**অর্থ:** আমি একবার বেইজিং গিয়েছিলাম।

**৪২. 会 (huì)**  
**Pinyin:** Huì  
**অর্থ:** পারা / জানা / মিটিং (can; be able to; meeting)  
**বাক্য:** 我会开车。  
**Pinyin:** Wǒ huì kāi chē.  
**অর্থ:** আমি গাড়ি চালাতে পারি।  
**বাক্য:** 我们在开会。  
**Pinyin:** Wǒmen zài kāi huì.  
**অর্থ:** আমরা মিটিং করছি।
