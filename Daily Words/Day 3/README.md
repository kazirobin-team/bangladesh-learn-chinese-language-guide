
**২৫. 多少 (duōshao)**  
**Pinyin:** Duōshao  
**অর্থ:** কত (how many; how much)  
**বাক্য:** 你们学校有多少学生？  
**Pinyin:** Nǐmen xuéxiào yǒu duōshao xuésheng?  
**অর্থ:** তোমাদের স্কুলে কতজন ছাত্র আছে?
