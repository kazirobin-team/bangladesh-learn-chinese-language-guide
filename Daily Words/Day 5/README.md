
**৪৮. 开 (kāi)**  
**Pinyin:** Kāi  
**অর্থ:** চালানো / খোলা / আয়োজন করা (drive; open; hold)  
**বাক্য:** 我开了三年出租车了。  
**Pinyin:** Wǒ kāi le sān nián chūzūchē le.  
**অর্থ:** আমি তিন বছর ধরে ট্যাক্সি চালাচ্ছি।  
**বাক্য:** 我爸爸开了一个小公司。  
**Pinyin:** Wǒ bàba kāi le yī gè xiǎo gōngsī.  
**অর্থ:** আমার বাবা একটা ছোট কোম্পানি খুলেছেন।  
**বাক্য:** 请开门，让我进去。  
**Pinyin:** Qǐng kāi mén, ràng wǒ jìnqù.  
**অর্থ:** দরজা খুলুন, আমাকে ভিতরে যেতে দিন।

**৪৯. 看 (kàn)**  
**Pinyin:** Kàn  
**অর্থ:** দেখা / পড়া (look; watch; read)  
**বাক্য:** 你看，那本书在桌子上呢。  
**Pinyin:** Nǐ kàn, nà běn shū zài zhuōzi shàng ne.  
**অর্থ:** দেখো, সেই বইটা টেবিলের উপর আছে।  
**বাক্য:** 我在看书呢。  
**Pinyin:** Wǒ zài kàn shū ne.  
**অর্থ:** আমি বই পড়ছি।

**৫১. 块 (kuài)**  
**Pinyin:** Kuài  
**অর্থ:** টাকা (RMB) / টুকরো (piece; Yuan)  
**বাক্য:** 这本书三十块钱。  
**Pinyin:** Zhè běn shū sānshí kuài qián.  
**অর্থ:** এই বইটা ত্রিশ টাকা।

**অর্থ:** এসো, এক টুকরো তরমুজ খাও।

**৫২. 来 (lái)**

**৫৪. 了 (le)**
