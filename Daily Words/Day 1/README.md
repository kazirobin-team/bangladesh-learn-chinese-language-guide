
**১. 爱 (ài)**  
**Pinyin:** Ài  
**অর্থ:** ভালোবাসা (love)  
**বাক্য:** 我爱你।  
**Pinyin:** Wǒ ài nǐ.  
**অর্থ:** আমি তোমাকে ভালোবাসি। (I love you.)

**২. 吃 (chī)**

**Pinyin:** Wǒ xǐhuan chī mǐfàn.  
**অর্থ:** আমি ভাত খেতে পছন্দ করি। (I like eating rice.)

**৮. 不客气 (bú kèqi)**  
**Pinyin:** Bú kèqi  
**অর্থ:** অভ্যর্থনা নয় / আপনাকে স্বাগতম (You’re welcome.)  
**বাক্য:** 甲：谢谢你！乙：不客气。  
**Pinyin:** Xièxie nǐ! — Bú kèqi.  
**অর্থ:** ধন্যবাদ! — আপনাকে স্বাগতম।
